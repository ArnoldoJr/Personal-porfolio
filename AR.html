// Resistor Scanner

The app decodes 4-band resistors using the phone camera and gets the value in ohms.

The resistors must be placed inside the square on a white non-reflective background as shown below.

There were several obstacles that I had to overcome to achieve this. One of them is lightning. The colour values can change drastically depending on the intensity and colour of the light. Despite this, I've been trying not to use any computer vision libraries from the start and this is what I’ve achieved so far:

The app also calculates resistor values by selecting its band colors on the grid. The colour bands of the desired resistor value can also be obtained by typing its value in ohms in the top-right corner.

I am currently working on supporting 5-band resistors as well!

// Network Sockets (C#)

A socket is one endpoint of a two-way communication link between two programs running on the network. In this project I created a Synchronous Server Socket that handles every incoming connection from clients in different threads.

Among its main functionalities it includes:

* Communication in both directions.
* Send message to either a single socket or to all sockets.
* Handling Client Disconnections.
* Shutting down server socket.
